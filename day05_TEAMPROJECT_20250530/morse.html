모스부호Morse code

  • 신호발생장치로 짧은 신호(·)와 긴 신호(-)를 적절히 조합하여 문자 기호를 표기하는 방식
  • 미국의 화가이자 발명가 새뮤얼 핀리 브리즈 모스가 1837년 고안
  • 1844년 최초로 미국의 볼티모어와 워싱턴 D.C. 사이 전신 연락에 사용

모스부호 규칙

  • 선(dash) 길이는 점(dot)의 3배일 것
  • 한 자(로마자 또는 한글자모 문자)를 형성하는 선과 점 사이 간격은 1점과 같을 것
  • 문자와 문자, 문자와 기호 사이 간격은 3점과 같을 것
  • 단어와 단어 사이는 7점과 같을 것
